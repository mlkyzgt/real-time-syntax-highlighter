# Real-Time Syntax Highlighter

**Real-Time Syntax Highlighter** is a dynamic, browser-based text editor designed to analyze the syntax of a simple programming language. It processes written code in real-time, instantly reports errors, and performs syntax highlighting.

---

## Key Features

- **Lexical Analysis**
  Scanning character sequences using the logic of **Regular Expressions** and **Finite Automata** to instantly tokenize them into meaningful building blocks.

- **Syntax Parsing**
  Real-time validation of the generated tokens against the rules of **Formal Languages** and the specific grammar of the defined programming language.

- **Dynamic Highlighting**
  Instantly colorizing variables, functions, numbers, and strings to significantly improve code readability.

- **Instant Error Reporting**
  Detecting and displaying syntax errors or incomplete expressions to the user simultaneously, without any delay.

- **Expression Analysis**
  Logical validation and evaluation of arithmetic operations, parenthesis precedence, and function calls.

---

## Supported Language Constructs

The editor currently recognizes and processes the following basic programming structures:

- **Data Types:** Numbers, Strings, and Booleans
- **Control Structures:** If-Else conditional statements
- **Loops:** Basic loop iterations
- **Variables:** Variable declarations and assignments
- **Operators:** Mathematical and logical expressions
- **Functions:** Custom function calls

## Technical Stack

This project is built with a pure (vanilla) approach, bringing core computer science fundamentals to the web without relying on external libraries.

- **User Interface:** A modern, fast, and clean design developed using strictly **HTML** and **CSS**.
- **Engine:** A custom-built lexer and parser developed entirely from scratch using **JavaScript**.
